Save all opened tabs in Chrome to Pocket.

This is not an oficial extension from Pocket, but it is based on oficial extension code!


HOW TO USE:
===========
In "src/js/ril.js" use your generated consumer key (http://getpocket.com/developer/) in "oAuthKey" variable.
